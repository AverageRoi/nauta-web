# Nauta Web

Website for Nauta, a free Discord bot for Minecraft servers.

Nauta helps communities store, find, and manage Minecraft coordinates directly from Discord.

## Links

- Website: https://nautabot.netlify.app/
- GitHub: https://github.com/AverageRoi

## Development

```sh
pnpm install
pnpm dev
```

## Build

```sh
pnpm build
pnpm preview
```
